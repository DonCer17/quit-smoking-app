# 🚭 StopSmoking

A progressive web app (PWA) to help you quit smoking gradually — built with vanilla JavaScript, HTML and CSS.

## 📱 Try it

👉 [Open StopSmoking](https://doncer17.github.io/quit-smoking-app/)

To install on iPhone: open the link in Safari → Share → Add to Home Screen

## 💡 Why I built this

I wanted to quit smoking and couldn't find an app that matched my approach — 
gradually increasing the time between cigarettes instead of following a rigid plan.
So I built one.

## ✨ Features

- **Personalized onboarding** — set your daily cigarettes, pack price and preferred method
- **Two modes** — structured 8-week plan or dynamic increasing intervals (+10 min per cigarette)
- **Daily tracker** — log each cigarette and compare with yesterday
- **Smart timer** — countdown to your next allowed cigarette
- **Statistics** — streak, 7-day chart, best day, money saved
- **Advanced stats** — unlocked after 7 days of real data
- **PWA** — installable on iPhone and Android, works offline

## 🛠️ Built with

- Vanilla JavaScript (no frameworks)
- HTML5 & CSS3
- Progressive Web App (Service Worker, Web Manifest)
- localStorage for data persistence
- Hosted on GitHub Pages
